# UI Component Library

再利用可能なUIコンポーネントライブラリ

## コンポーネント
- AdventureCard - 公募情報カード
- QuestBoard - 掲示板レイアウト
- HeroButton - CTAボタン
- ProgressBar - 進捗表示
- NotificationBell - 通知システム

## デザインシステム
- カラーパレット: 冒険者テーマ
- タイポグラフィ: 読みやすさ重視
- アイコン: RPG風アイコンセット
- アニメーション: 微細な動き

## Storybookサポート
```bash
npm run storybook
```
